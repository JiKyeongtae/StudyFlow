# 1. 프로젝트 수행 목적

## 1.1 프로젝트 정의

시간표, 일정, 과제, 시험 일정을 통합적으로 관리할 수 있는 대학생 학습 관리 애플리케이션

---

## 1.2 프로젝트 배경

대학생들은 수업 시간표, 과제, 시험 일정 등을 여러 플랫폼이나 메모장에 따로 기록하는 경우가 많다.

이러한 방식은 중요한 일정을 놓치거나 과제 제출 기한을 잊어버릴 가능성이 있으며, 학업 계획을 체계적으로 관리하기 어렵다는 문제점이 있다.

본 프로젝트는 대학생들이 학업과 관련된 정보를 하나의 애플리케이션에서 효율적으로 관리할 수 있도록 지원하기 위해 개발하였다.

---

## 1.3 프로젝트 목표

* 시간표 관리 기능 제공
* 일정 등록 및 관리 기능 제공
* 과제 등록 및 진행률 관리 기능 제공
* 시험 일정 등록 및 D-Day 계산 기능 제공
* 학습 정보를 한 화면에서 확인할 수 있는 대시보드 제공

---

# 2. 프로젝트 개요

## 2.1 프로젝트 설명

StudyFlow는 대학생의 학습 관리를 지원하는 iOS 애플리케이션이다.

사용자는 시간표, 일정, 과제, 시험 정보를 등록할 수 있으며, 등록된 정보는 UserDefaults를 통해 저장되어 앱을 종료한 후에도 유지된다.

홈 화면에서는 오늘 수업 수, 남은 과제 수, 과제 진행률, 오늘 일정, 가장 가까운 시험 정보를 한눈에 확인할 수 있다.

## 2.3 주요 기능

### 홈 화면

* 현재 날짜 표시
* 시간대별 인사말 표시
* 오늘 수업 수 표시
* 남은 과제 수 표시
* 과제 진행률 표시
* 오늘 일정 표시
* 다가오는 시험 D-Day 표시

### 일정 관리

* 일정 등록
* 일정 삭제
* 일정 저장

### 과제 관리

* 과제 등록
* 과제 완료 체크
* 과제 삭제
* D-Day 계산

### 시험 관리

* 시험 일정 등록
* 시험 삭제
* D-Day 자동 계산

### 시간표 관리

* 요일별 수업 등록
* 시작 시간 및 종료 시간 설정
* 시간표 시각화
* 수업 삭제
